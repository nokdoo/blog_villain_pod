NAME

Benchmark - 실행시간 벤치마크

SYNOPSIS

        use Benchmark qw(:all) ;

        timethis ($count, "code");

        # Use Perl code in strings...
        timethese($count, {
                'Name1' => '...code1...',
                'Name2' => '...code2...',
        });

        # ... or use subroutine references.
        timethese($count, {
                'Name1' => sub { ...code1... },
                'Name2' => sub { ...code2... },
        });

        # cmpthese can be used both ways as well
        cmpthese($count, {
                'Name1' => '...code1...',
                'Name2' => '...code2...',
        });

        cmpthese($count, {
                'Name1' => sub { ...code1... },
                'Name2' => sub { ...code2... },
        });

        # ...or in two stages
        $results = timethese($count,
                {
                        'Name1' => sub { ...code1... },
                        'Name2' => sub { ...code2... },
                },
                'none'
        );
        cmpthese( $results ) ;

        $t = timeit($count, '...other code...')
        print "$count loops of other code took:",timestr($t),"\n";

        $t = countit($time, '...other code...')
        $count = $t->iters ;
        print "$count loops of other code took:",timestr($t),"\n";

        # enable hires wallclock timing if possible
        use Benchmark ':hireswallclock';

DESCRIPTION

Benchmark 모듈은 함수들의 실행시간이 얼마나 걸리는지에 대한 루틴들을 캡슐화한다.

timethis

코드를 여러 번 실행.
