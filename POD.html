NAME

POD(Plain Old Documentation)

DESCRIPTION

perl진영의 문서파일 pod에 대해 살펴본다.

이 문서는 pod2html를 이용해 pod파일을 html로 변환시킨 결과물이다. (css, js 추가 등 별도의 작업 필요)

COMMANDS

명령어 앞줄과 뒷줄에 공백을 넣어주자.

=pod, =cut

각각 pod문서의 시작과 끝을 알린다.

=head1 text, =head2 text, =head3 text, =head4 text

문단 사이에 삽입되어 제목을 나타낸다. 숫자가 커질수록 제목의 크기가 점점 작아진다. 숫자의 최대값은 4다.

문서를 작성할 때 head1 다음 head2를 건너뛰지 않도록 한다. (인덱스가 깨져버림)

=over, =item text, =back

=over와 =back은 리스트의 시작과 끝을 알린다.

=item text text를 이름으로 하는 요소를 생성한다. 첫 번째 요소의 text자리가 * 로 시작하면 순서가 없는 리스트를 구성하게 된다.

ex)

 =over

 =item * text1 

 =item * text2

 =back
 
=====>>>>>

  • text1

  • text2

=begin format, =end format, for targetname text

pod를 벗어나 새로운 format영역을 만든다. 그 내용은 =begin과 =end 사이 또는 =for의 text부분에 들어간다. 둘의 차이는 적용범위다.

(format은 text/data를 나타낸다고 하는데.. html외의 다른 포맷은 모르겠다...)

=encoding encodingname

pod에 적용할 인코딩을 선택한다. 전체 pod에 적용되니 한 번만 입력하면 된다.

pod의 기본 인코딩은 US-ASCII이기 때문에 한글이 깨져버린다. =encoding utf8을 사용하면 해결된다.

Formatting

텍스트에 볼드, 링크 등의 추가적인 효과를 넣어준다.

I<text>, B<text>, C<text>, L<text|scheme:...>, E<escape>

각각 italic, bold, code, link, < (=E<escape>)을 나타낸다.

이 외의 formatting은 여기에서 확인할 수 있다.
