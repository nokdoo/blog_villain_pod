NAME

DBIx::Class::Schema::Config - DBIx::Class에 대한 자격 증명 관리.

DESCRIPTION

DBIx::Class::Schema::Config는 DBIx::Class::Schema의 하위 클래스로서, 파일로부터 자격 증명 및 설정을 로드할 수 있다. 실제 코드는 설정 파일에서 사용되는 이름만 알면 된다. 이것은 데이터베이스 자격 증명을 관리하는 것을 간단하게 만든다.

이미 사용하고 있는 DBIx::Class::Schema를 변환하여 DBIx::Class::Schema::Config를 사용하는 예제는 http://www.symkat.com/credential-management-in-dbix-class에서 확인할 수 있다.

SYNOPSIS

/etc/dbic.yaml

        MY_DATABASE:
                dsn: "dbi:Pg:host=localhost;database=blog"
                user: "TheDoctor"
                password: "dnoPydoleM"
                TraceLevel: 1

My/Schema

        package My::Schema
        use warnings;
        use strict;
         
        use base 'DBIx::Class::Schema::Config';
        __PACKAGE__->load_namespaces;

My/Code

        package My::Code;
        use warnings;
        use strict;
        use My::Schema;
         
        my $schema = My::Schema->connect('MY_DATABASE');
