NAME

ctags - tag index 파일 생성

SYNOPSIS

ctags [option] [file(s)] etags [option] [file(s)]

DESCRIPTION

ctags, etags 는 tag index 파일을 생성한다. tag index 파일은 에디터가 개체의 위치를 쉽고 빠르게 찾을 수 있게 한다. 하나의 개체를 tag 라고 한다.

ctags 는 다양한 개체의 정보를 나타내는 파일을 생성할 수 있다.

tag index 파일은 많은 에디터로부터 지원된다. 이것은 사용자가 해당 개체의 파일, 라인 번호로 이동할 수 있게 한다.

ctags 는 다른 언어에 대해 다른 tag 를 생성할 수 있다. tag 의 종류와 이름이 각각 생성된다. "--list-languages", "--list-kinds" 옵션을 보자.

SOURCE FILES

"--language-force" 옵션이 지정되지 않았다면, 파일명을 따라 언어를 자동적으로 선택한다. "--list-maps" 옵션을 통해 매필되는 언어를 표시할 수 있으며, "--langmap" 옵션을 통해 변경 가능하다.
